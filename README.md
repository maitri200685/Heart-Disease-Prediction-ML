# ❤️ Heart Disease Prediction Web Application

An end-to-end Machine Learning web application that predicts the likelihood of heart disease based on patient health information. The project combines data analysis, machine learning, and a user-friendly Streamlit interface to provide real-time predictions.

---

## 📖 Project Overview

Heart disease is one of the leading causes of death worldwide. Early prediction can help patients seek timely medical attention.

This project uses Machine Learning algorithms to analyze patient medical data and predict whether a person is likely to have heart disease. The application is deployed through an interactive Streamlit interface where users can input their medical information and receive instant predictions.

---

## 🚀 Features

- Interactive Streamlit web interface
- Real-time heart disease prediction
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature Encoding and Scaling
- Comparison of multiple Machine Learning models
- Trained K-Nearest Neighbors (KNN) model for prediction

## 📊 Dataset Features

The model uses the following patient information:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise-Induced Angina
- Oldpeak
- ST Slope

---

## 🤖 Machine Learning Models Evaluated

The following algorithms were trained and compared:

| Model | Accuracy |
|--------|----------|
| Logistic Regression | **86.96%** |
| K-Nearest Neighbors (KNN) | **86.41%** |
| Naive Bayes | **85.33%** |
| Support Vector Machine (RBF) | **84.78%** |
| Decision Tree | **78.80%** |

> The trained KNN model is used for predictions in the web application.
